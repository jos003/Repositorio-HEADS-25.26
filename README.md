# Repositorio-HEADS-25.26
Repositório do programa doutoral em Ciência de Dados em Saúde (HEADS) da Faculdade de Medicina da Universidade do Porto (FMUP). Este espaço destina-se ao apoio colaborativo entre os doutorandos, promovendo a partilha de conhecimento e o desenvolvimento de competências em ciência de dados aplicada à saúde.
